# Profile API Documentation

## Endpoint
`GET /api/profile`

## Purpose
Fetches the authenticated user's profile details (name, email, role, etc.).

## Dependency
Requires a valid session token obtained from the **Login API**.

## Request
```http
GET /api/profile HTTP/1.1
Host: api.example.com
Authorization: Bearer <session_token>
